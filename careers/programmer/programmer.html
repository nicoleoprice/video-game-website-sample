Types of Programmers

    There are several different types of programming jobs within the game industry. Programmers are also referred to as game developers, they are used interchangably sometimes. The programmer's role is to add behavior onto the things that the artists make, like making a character walk for example. Below are some of the types of programmers.

Audio Programmer
An audio programmer works alongside the sound designer to produce the audio in the game. This job entails designing and writing the code that would trigger sound effects along with other interactive audio, such as much and speech.
Graphics Programmer
This type of programmer uses code to get the visuals within the applications and game to display the way the creators want them to. These programmers also try to program them in a way to make them seem lifelike and believable.
Network Programmer
This type of programmer writes a code that allows players to play against each other or play together by connecting through a computer network or the Internet. This type of programming is one of the most challenging in creating a video game.
